## 🚀 Cómo ejecutar el proyecto localmente

Este proyecto está construido con **Astro**.

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

3. **Construir para producción:**
   ```bash
   npm run build
   ```
